## Socialmedia App React.js

Masuk kedalam direktori projek, jalankan perintah:

### `npm install`

Menjalankan mode development:
### `npm start`

Menjalankan mode production:
### `npm run build`
### `npm install -g serve`
### `serve -s build`

## Dependencies
- Redux toolkit
- Axios
- Daisy UI & Tailwind CSS
- react-alert
- lottie-react
